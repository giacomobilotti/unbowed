# OxCal Models

This folder contains the OxCal code used to refine site chronologies.

The models were run using OxCal 4.4 (Bronk Ramsey 2009). Results are presented in Table 1 of the paper, with additional details provided in Sections 1–2 of the Supplementary Information.

The code incorporates stratigraphic relationships and the published model for Cutamalla (Mader et al. 2024). In addition, it updates the published model for Jauranga (Osborn et al. 2023) by incorporating 12 additional radiocarbon dates and more detailed stratigraphic relationships, and presents a new model for Collanco.

The calibration curve is a bespoke mixture of SHCal20 (Hogg et al. 2020) and IntCal20 (Reimer et al. 2020), following estimates of the proportion of Northern Hemisphere air present at Cutamalla over the last 80 years (Ancapichún et al. 2022; Mader et al. 2024).

See the main text and SI for the full references.
